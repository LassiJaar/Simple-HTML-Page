# 🔴 SimpléDex
A simple application for looking up information on different Pokémon! Project is powered by [PokéAPI](https://pokeapi.co/). Built with simple HTML and JavaScript functions to deliver a simple census on any Pokémon!

## 🛠️ Usage
1. Enter the desired ID or name
2. Press "GO!"

## ✨ Information available
* ID and Name
* Images
* Types
* Description
* Stats
